# Schema change

Użyj tej komendy, gdy zadanie dotyczy zmian w SQLite, plikach SQL albo modelu danych.

## Procedura

1. Przeczytaj `.agents/skills/schema-change/SKILL.md`.
2. Ustal wpływ na istniejące dane i odczyty/zapisy.
3. Dowieź minimalną zmianę z testami i dokumentacją.
4. Uruchom `uv run poe check`.
