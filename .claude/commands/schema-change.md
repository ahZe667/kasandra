# Schema change

Użyj tej komendy, gdy zadanie dotyczy zmian w SQLite, plikach SQL albo modelu danych.

## Procedura

1. Przeczytaj `AGENTS.md`.
2. Przeczytaj `.agents/skills/schema-change/SKILL.md`.
3. Ustal wpływ na istniejące dane i odczyty/zapisy.
4. Dowieź minimalną zmianę z testami i dokumentacją.
5. Uruchom `powershell -ExecutionPolicy Bypass -File scripts/repo-check.ps1`.

## Input użytkownika

$ARGUMENTS
