# Docs sync

Użyj tej komendy do synchronizacji dokumentacji i przykładów po zmianie kodu lub założeń.

## Procedura

1. Przeczytaj `AGENTS.md`.
2. Przeczytaj `.agents/skills/docs-sync/SKILL.md`.
3. Zidentyfikuj dokumenty będące źródłami prawdy dla zmienianego obszaru.
4. Zaktualizuj treść zachowując spójność między dokumentami.
5. Uruchom `powershell -ExecutionPolicy Bypass -File scripts/repo-check.ps1`.

## Input użytkownika

$ARGUMENTS
